# CLI overview

Use these commands most often:

- `dbt-osmosis yaml refactor`: organize YAML files and inherit docs in one pass
- `dbt-osmosis yaml organize`: create/move YAML files based on routing rules
- `dbt-osmosis yaml document`: sync column docs, tags, and metadata
- `dbt-osmosis sql run`: execute ad-hoc SQL with dbt context
- `dbt-osmosis workbench`: launch the Streamlit workbench
